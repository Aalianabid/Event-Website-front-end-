
## 🛠️ Tech Stack

* **HTML5** – Semantic structure
* **CSS3** – Styling, animations, responsive design
* **JavaScript (ES6+)** – Dynamic rendering & interactivity
* **CSS Grid & Flexbox** – Responsive layouts

* 📊 **Live Stats Dashboard** – Events, attendees, locations (auto-updated)
* 📅 **Sort by Date** – Toggle chronological view with active filters
* 👥 **Attendee Count** – Visible on each event card
* 📱 **Fully Responsive** – Mobile-first across all screen sizes
* 🎨 **Premium Animations** – Fade, slide, shimmer, ripple, hover effects
* 🎚️ **Custom Scrollbar** – Styled to match theme
